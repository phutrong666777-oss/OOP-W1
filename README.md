# OOP Week 1 - Basic Classes

## Description

This project implements basic Object-Oriented Programming concepts in C++.
Include class of Fraction, Point, Rectangle.

## Features

### Fraction (Question1)

- Input / Output
- Addition
- Subtraction
- Multiplication
- Division
- Simplification

### Point (Question2)

- Input / Output
- Distance calculation

### Rectangle (Question3)

- Rectangle validation
- Perimeter calculation
- Area calculation
- Draw UML diagram
